# 喝口水

一个零依赖的饮水提醒与记录 demo。打开页面即可记录今日饮水量、设置每日目标、开启浏览器提醒，并查看最近 7 天的饮水趋势。

## 本地使用

直接打开 `index.html` 可以使用记录功能。若要启用 PWA 缓存和更稳定的浏览器通知，建议用本地服务运行：

```bash
python3 -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 功能

- 快速记录 100 / 200 / 300 / 500 ml，也支持自定义饮水量
- 今日目标、进度条、动态水杯进度
- 今日记录时间线、撤销上一条、清空今日
- 提醒间隔、提醒时段、每日目标设置
- 每周目标和周日奶茶奖励进度
- 页面内提醒和浏览器通知
- 最近 7 天趋势、7 日平均、连续达标天数
- 本地保存和 JSON 导出

## 说明

数据保存在当前浏览器的 `localStorage` 中，不会上传到服务器。提醒需要页面保持打开；浏览器通知需要用户授权。
